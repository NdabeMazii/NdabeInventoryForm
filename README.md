# Web Automation Advance - Inventory Form Testing

## Project Overview

This Java-based automated testing project validates an advanced web automation inventory form. The test suite covers complete end-to-end workflows including device selection, configuration, order placement, and invoice management.

## Test Scenario Coverage

### Main Test Flow
- **Login**: Authentication with test user credentials
- **Device Selection**: Phone device type selection
- **Brand Configuration**: Apple brand selection with dropdown validation
- **Device Preview**: Visual confirmation of selected device
- **Storage Selection**: 128GB storage option with price calculation (R480.00)
- **Color Selection**: Blue color application
- **Quantity Entry**: Multiple unit ordering (Qty: 2, Subtotal: R960.00)
- **Address Input**: Shipping address entry ("123 Test Street")
- **Shipping Options**: Express Shipping selection (R25.00)
- **Warranty Selection**: 1-year warranty addition (R49.00)
- **Discount Application**: SAVE10 coupon code (10% off)
- **Purchase Confirmation**: Success notification with order details
- **Invoice Management**: View, access, and validate invoice history
